　神々が統治する世界ブリトデネク…のある場所にある都市ロミア。
この街ではとある事件が起きているようです。
謎の存在に街へ飛ばされたあなたはこの事件を解決できるのか？

●詳細説明
・クリア推定時間は2～6時間程度です。
・感想などやアドバイス、その他何かございましたらこちらのフォーラムまたは私のX(https://x.com/gol559)また本ワールドのトレーラーのコメント欄(https://youtu.be/0fPHpwLvCuY)までお願いします。
・マルチでプレイする際には、同梱しているサーバープロパティーを使用してください。(サーバーリソースパックが使用できます)

◎必須設定
・バージョンは1.21.8です(1.21.7以下及び1.21.9以上のバージョンで入ってしまった際は、お手数ですがワールドを入れなおしてください。)
・パーティクルの表示：すべて
・言語設定：日本語または英語

○推奨設定(任意でOKです)
・プレイ人数：１～5人程度(大人数でもプレイ可能ではありますが、ボスが強くなるのでおすすめはしません)
・明るさ：お好みで(明るい方が良いと思います)
・主音量：聞こえる程度にお好みで(オフにはしないでください)
・描画距離：お好みで(描画距離に問わず動作するよう設計しましたが、描画距離が少ないと正常に動作しない可能性があります。なお、デバッグは8チャンクで行いました。)
・MODを一切入れないバニラ環境でのプレイ(MODを入れて動く想定で作っていないため、おすすめしません。)

○マルチプレイ時推奨設定
・サーバープロパティに以下の項目を記入してください

●難易度について
・Easy：どなたでも快適な難易度です。
・Normal：推奨難易度です。
・Hard：様々な点で難しくなります。
・Very Hard：玄人向けの難易度です。
・Nightmare：クリア想定をしていません。

難易度設定はワールド内で変更可能です。
コマンド等では絶対に設定しないでください。



●困った時のQ&A
Ｑ：ワールドに入ろうとすると警告がでるんだけど?
Ａ：バイームをいじっているために警告がでます。無視しても問題ありません。

Ｑ：ワールドの導入方法を教えてください。
Ａ：こちらの動画を参照してください( https://youtu.be/t2xBxEWfj2s )

Ｑ：杖？宝具？何これ？
Ａ：このワールドでは、コマンドによって、魔法や剣術、弓術等のシステムがあります。

Q:魔法が効かない敵がいるんだけど？
Ａ:本ワールドには一部魔法があまり効かない敵モブも存在します。
　 逆に物理攻撃がほとんど効かない敵モブも存在します。

Ｑ：ベッドとかって意味あるの？
Ａ：このワールドでは睡眠をとるとボーナスがつくようになっています。

Ｑ；スターシロップを飲んだらデバフ効果が消えないんだけど…。
Ａ：スターシロップのデバフ効果は大地の雫を飲まない限り消えません。

Ｑ：お金が足りない！
Ａ：ダンジョンに潜るか街にいる人の相談を聞いてみましょう。また、釣りをしてみると…？

Ｑ：○○って不具合？仕様？
Ａ：配布場所または、私のX(@gol559)まで連絡いただければ、可能な限り対応いたします。
　　不具合報告の際はバージョンやシングルまたはマルチプレイか、などの情報を教えていただけると助かります。

Ｑ：バージョン1.21.7以前または1.21.9以降じゃできなの？
Ａ：補償対象外となりますのでプレイしないようにお願いいたします。

Ｑ：動画・生配信に使いたいんだけど？
Ａ：動画や生配信はご自由に行っていただいて構いません。
